Claude Code와 Codex 토큰 사용량을
잔디로 시각화하세요

GitHub 스타일 히트맵으로 Claude Code와 Codex 토큰 사용 패턴을 한눈에 파악하세요. 프라이버시를 지키면서, 메뉴바에서 바로.

Download GitHub
$ brew install --cask wheon06/tap/token-jandi
Features

당신의 Claude 사용 습관을 한눈에

🟩

잔디 히트맵

GitHub 기여 그래프처럼 20주간의 토큰 사용량을 색상 강도로 시각화합니다.

📊

상세 차트

일별·월별 사용량 차트로 토큰 소비 추이를 분석하고 패턴을 파악하세요.

🔥

스트릭 추적

연속 사용일, 오늘 사용량, 주간 합계 등 핵심 통계를 한눈에 확인합니다.

⚡

메뉴바 상주

별도 앱 창 없이 메뉴바 아이콘 클릭 한 번으로 즉시 확인할 수 있습니다.

🔄

자동 새로고침

1분~30분 간격으로 자동 갱신되어 항상 최신 사용량을 보여줍니다.

🌐

다국어 지원

한국어와 영어를 지원하며, 시스템 언어를 자동으로 감지합니다.

Install

3단계로 시작하세요

1

Homebrew로 설치

한 줄의 명령어로 간단하게 설치할 수 있습니다.

brew install --cask wheon06/tap/token-jandi
2

폴더 접근 허용

~/.claude와 ~/.codex 데이터를 찾을 수 있도록 홈 폴더 접근을 허용해주세요.

3

잔디 확인

메뉴바에 나타난 잔디 아이콘을 클릭하면 사용량 히트맵을 바로 볼 수 있습니다.

🔒

프라이버시 최우선

Token Jandi는 어떤 데이터도 외부로 전송하지 않습니다. 모든 분석은 로컬에서 이루어집니다.

✓ 데이터 수집 없음 ✓ 서드파티 SDK 없음 ✓ 로컬 전용 분석 ✓ 오픈소스 (MIT)

지금 시작하세요

Claude Code나 Codex를 사용하고 있다면, 당신의 토큰 사용 패턴을 확인해보세요.
